# 잉글리시몬 ⚡ (English-mon)

초등학교 1학년 아이를 위한 영어 단어 학습 게임입니다. 파일 하나(`index.html`)로 동작하며, 브라우저에서 바로 실행됩니다.

**▶ 플레이: https://ian939.github.io/english-mon/**

## 게임 방법

1. **단어 퀴즈 / 끝말잇기 / 카드 짝맞추기**로 영어 단어를 맞추면 🪙 코인을 벌어요 (첫 시도 정답 1개 = 1코인, 5콤보마다 보너스)
2. 상점에서 몬스터볼을 사요 — 몬스터볼 🔴 4코인(+30%) · 슈퍼볼 🔵 7코인(+40%) · 하이퍼볼 🟡 10코인(+60%) · 마스터볼 🟣 20코인(+80%)
3. **모험 출발!** 야생 포켓몬(일반/레어/유니크/전설)을 만나 볼을 던져요 — 볼마다 잡을 확률이 표시돼요
4. 잡은 포켓몬은 **도감(1~3세대, 386마리)**에 등록! 다 모아보세요 👑

## 특징

- 영유~초등 저학년 필수 영어 단어 **504개, 20개 카테고리** (모든 단어에 그림 이모지)
- **부모 모드**: 홈 오른쪽 위 👶/🧑 버튼으로 전환 — TOEFL(500단어) 또는 IELTS(레벨 1·2·3 × 150단어, 기본 Lv2) 중 선택, 힌트 없는 하드모드. 코인·볼·도감은 아이와 함께 모아요
- 영어 발음 자동 재생 (브라우저 TTS)
- 진행 상황 자동 저장 (같은 브라우저에서 이어하기)
- 설치·서버 불필요 — `index.html` 하나로 완결
